# Okafuta - Merchant Dashboard

**Transact** is a platform for merchants to manage their sales transactions and payout history.  

## Features
- **Transaction History**: Merchants can view all customer payments made to their account.  
- **Payout History**: Track all withdrawals and disbursements sent from Transact to the merchant’s bank or wallet.  
- **Simple Dashboard**: Easy-to-use interface for monitoring sales and payouts.  

## Tech Stack
- **Frontend**: React
